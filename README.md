# lift-tracker
LIFT - is my little powerlifting app. I use it to track my workouts and plan my exercises. 

Features:
- Enter 1RM for squats, bench presses, and deadlifts
- Import and export workout data to and from Excel
- Lightweight app built with HTML and JavaScript as a PWA

Installation:

On your phone, open this URL (https://raytiw.github.io/lift-tracker/lift_tracker.html) in Safari (iOS) or Chrome (Android) and:

iOS: Share icon → “Add to Home Screen”
Android: Menu → “Install App” or “Add to Home Screen”

The app will then also work offline (Service Worker caches everything).
